# Web Scraping Using Python

A collection of web scraping practice scripts using various Python libraries. This repository demonstrates different approaches to web scraping, from basic HTML fetching to advanced job portal scrapers.

## Overview

This repo contains practice scripts for learning web scraping techniques using popular Python libraries:
- **Requests** - Basic HTTP requests
- **BeautifulSoup** - HTML parsing
- **Requests-HTML** - JavaScript rendering

## Installation

1. Clone the repository:
```bash
git clone https://github.com/KaushallB/Web-Scraping-Using-Python.git
cd Web-Scraping-Using-Python
```

2. Install required packages:
```bash
pip install requests beautifulsoup4 requests-html 
```

3. Install Playwright browsers (for Crawl4AI):
```bash
playwright install
```
## Key Features Learned

1. **HTTP Requests** - Fetching web pages
2. **HTML Parsing** - Extracting data from HTML
3. **Regex Patterns** - Matching job listings and links
4. **Async Programming** - Faster concurrent scraping
5. **File Management** - Organizing scraped data
6. **Error Handling** - Managing failed requests
7. **Data Formatting** - Clean, readable output files
